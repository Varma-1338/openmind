# OpenMind: Your Personal AI Guide to Lifelong Learning

OpenMind is a web application that transforms any user interest into a personalized, structured learning journey. Powered by generative AI, it dynamically creates daily topics, curates relevant reading materials, and generates micro-quizzes to test comprehension.

## Key Features

- **AI-Powered Journey Creation:** Simply enter your interests, and OpenMind builds a complete, day-by-day learning plan.
- **Curated Content:** For each topic, the AI finds and explains relevant articles and resources.
- **Interactive Quizzes:** Test your knowledge with automatically generated quizzes at the end of each daily session.
- **Personalized Experience:** Track your progress and maintain a daily learning streak.
- **Pre-generated Courses:** Explore curated learning paths in various professional and academic fields.
